# Job4j_Cinema
## Учебный проект курса **[Job4j](https://job4j.ru/)**, раздела "Web", уровня "Middle".
### Клиент-серверное приложение имитирующее сервис покупки билетов в кинотеатр.
### В проекте используется:
   - Spring Boot
   - Bootstrap,
   - Thymeleaf,
   - JDBC,
   - Liquibase
___
### Тестирование:
   - H2 Database
   - Mockito
   - Junit
   - Hamcrest
___
